# Robot-Planar-2GDL-simulacion
Este proyecto corresponde a la simulación dinámica de un robot manipulador planar de dos grados de libertad (2 GDL) El modelo dinámico fue obtenido mediante el formalismo de Euler-Lagrange, considerando:  - Matriz de inercia (M) - Matriz de Coriolis (C) - Vector de gravedad (g) - Modelo de fricción - Torques aplicados por los motores
